## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Forms**: React Hook Form
- **Calendar**: React Calendar
- **Maps**: Google Maps API
- **Email**: EmailJS or Resend
- **Images**: Next.js Image + Unsplash
- **Deployment**: Vercel

2. Add items to cart (with customizations)
3. Review order
4. Enter delivery/pickup details
5. Payment (or cash on delivery)
6. Confirmation
